This repository contains my web portfolio developed for GEOG 3540. The site showcases a collection of cartographic projects, including choropleth, point symbol, isarithmic, bivariate, and flow mapping assignments.

The portfolio is built using HTML, CSS, and Bootstrap, and is hosted using GitHub Pages. It includes a dedicated flow map page featuring an internal migration flow map of Rwanda, along with design justification and interpretation of flow patterns.

The website also includes a short biography, a downloadable resume, and links to interactive Observable notebooks for each mapping assignment.
